# Feature 11: Policy UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wrap `/policy` in a Jinja template that displays `policy.yaml` in a readable form (raw + parsed). Also retroactively add `static/app.js` for `/audit` polling (was deferred from F10; per spec section 3.7).

**Architecture:** New `templates/policy.html` extends `base.html`. The `/policy` route returns `TemplateResponse` with `content` (raw YAML), `parsed_rules` (list of `{tool, decision, reason, when}`), and `agent` (if specified). The audit polling script in `static/app.js` is added to `audit.html` via `<script>` in `base.html` (so it runs on every page; it only acts when `#audit-table` is present).

**Tech Stack:** `jinja2`, vanilla JS (no framework). No new deps.

## Contract

**`GET /policy`** (HTML) → renders `policy.html` with:
- `agent` (str | None): the `agent:` field of the policy, if any
- `content` (str): raw YAML
- `rules` (list of dicts): parsed rules `{tool, decision, reason, when}`
- `default` (str): default decision

**Missing policy file** → 500 with a clear HTML error page (not JSON).

### Task 3: Add audit polling JS (`static/app.js`)

**Files:**
- Create: `static/app.js`

- [ ] **Step 1: Create the polling script**

```js
// Polls /api/audit every 3s when on the audit page; updates the table
// and the chain-valid banner. No framework; no dependencies.
(function () {
  const REFRESH_MS = 3000;

  function escapeHtml(s) {
    return String(s)
      .replace(/&/g, "&amp;")
      .replace(/</g, "&lt;")
      .replace(/>/g, "&gt;")
      .replace(/"/g, "&quot;")
      .replace(/'/g, "&#39;");
  }

  function renderEventRow(e) {
    const ts = escapeHtml(e.ts);
    const agent = escapeHtml(e.agent_id);
    const action = escapeHtml(e.action);
    const decision = escapeHtml(e.decision);
    const reason = escapeHtml(e.reason || "");
    const rid = escapeHtml((e.request_id || "").slice(0, 12) + "\u2026");
    const hash = escapeHtml((e.this_hash || "").slice(0, 12) + "\u2026");
    return (
      "<tr>" +
      '<td class="ts">' + ts + "</td>" +
      "<td>" + agent + "</td>" +
      "<td><code>" + action + "</code></td>" +
      '<td><span class="badge badge-' + decision + '">' + decision + "</span></td>" +
      '<td class="reason">' + reason + "</td>" +
      '<td class="rid"><code>' + rid + "</code></td>" +
      '<td class="hash"><code>' + hash + "</code></td>" +
      "</tr>"
    );
  }

  function start() {
    const tableBody = document.querySelector(".audit-table tbody");
    const banner = document.querySelector(".banner");
    if (!tableBody || !banner) return;

    function refresh() {
      fetch("/api/audit", { headers: { "Accept": "application/json" } })
        .then(function (r) { return r.json(); })
        .then(function (body) {
          const events = (body.events || []).slice().reverse();
          if (events.length === 0) {
            tableBody.innerHTML = "";
          } else {
            tableBody.innerHTML = events.map(renderEventRow).join("");
          }
          if (body.chain_valid) {
            banner.className = "banner banner-ok";
            banner.textContent = "Chain valid · " + events.length + " event(s)";
          } else {
            banner.className = "banner banner-error";
            banner.textContent = "CHAIN TAMPERED — investigate immediately";
          }
        })
        .catch(function () { /* ignore transient errors */ });
    }

    refresh();
    setInterval(refresh, REFRESH_MS);
  }

  if (document.readyState === "loading") {
    document.addEventListener("DOMContentLoaded", start);
  } else {
    start();
  }
})();
```

### Task 4: Modify `app.py` to render the policy page

**Files:**
- Modify: `app.py`

- [ ] **Step 1: Replace the `/policy` route with a template-rendering version**

Find:
```python
    @app.get("/policy")
    async def policy_view() -> dict[str, str]:
        if not cfg.policy_path.exists():
            raise HTTPException(
                status_code=500, detail=f"policy file not found: {cfg.policy_path}"
            )
        return {"content": cfg.policy_path.read_text()}
```

Replace with:
```python
    @app.get("/policy", response_class=HTMLResponse)
    async def policy_view(request: Request) -> Response:
        if not cfg.policy_path.exists():
            return HTMLResponse(
                f"<html><body><h1>500</h1><p>policy file not found: {cfg.policy_path}</p></body></html>",
                status_code=500,
            )
        import yaml as _yaml

        raw = cfg.policy_path.read_text()
        try:
            parsed = _yaml.safe_load(raw) or {}
        except _yaml.YAMLError as exc:
            return HTMLResponse(
                f"<html><body><h1>500</h1><p>invalid YAML: {exc}</p></body></html>",
                status_code=500,
            )
        return templates.TemplateResponse(
            request,
            "policy.html",
            {
                "agent": parsed.get("agent"),
                "content": raw,
                "rules": parsed.get("rules") or [],
                "default": parsed.get("default", "deny"),
            },
        )
```

Note: `import yaml` inside the function avoids making yaml a hard module-level import (it's already imported transitively by `zta.policy`, but importing it explicitly here makes the dependency explicit).
